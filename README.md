# Betta Trading Landing Page

Landing page untuk layanan copy trading dan Expert Advisor berbasis Next.js App Router.

## Menjalankan proyek

- Install dependensi: `npm install`
- Jalankan dev server: `npm run dev`
- Build produksi: `npm run build`
- Jalankan produksi: `npm start`

## Catatan

Konten landing page berfokus pada transparansi kontrol dana, model bagi hasil, dan risiko.
